1 команда) git		
2 команда) git init
2.1 команда)(не обязательна служит проверкой статуса) git status
3 команда)(создание репозитория(следим за файлом хтмл) git add index.html 
3.1команда)(чтобы добавить ВСЕ новые фалы и изменения) git add .

4 команда)(создание коммита) git commit -m "fits commit"

5 команда)(создаем папку в которую засовываем фалы которые не хотим показывать).gitignore

6 команда)(создание ветки) git branch <название ветки>
6.1 команда)(проверка ветки) git branch

6.2 команда)(удалить ненужную ветку) git branch -D <название ветки>
6.3 команда)(перейти в другую ветку) git checkout <название ветки>

6.4 команда)(создание ветки и сразу перейти на нее) git checkout -b <название ветки>


7 команда)(обьединение веток)git merge <название ветки>

8 команда)(на сайте гита создаем репозиторий от туда копируем такую похожую строку) git remote add origin https://github.com/WeeRbe/Git-course.git

8.1 команда)(так же копируем строку ниже строки в 8 команде) git push -u origin <название ветки>


9 команда)(если создаем новые изменения и надо залить их на гит) git push
